ID选择器

通过元素的id属性来选取元素，选择器以 # 号开头并接上元素的id名称：

在整个HTML文档中，一个元素要么没有id，要么id属性的值是唯一的。id属性就类似于现实世界的身份证号，不同的人名字可能是相同的，但身份证号一定是不同的。所以，通过ID选择器可以选择一个指定的、唯一的元素来添加样式。

不过，你在编写HTML文档时，可以将多个元素的id属性赋予相同的值，浏览器并不会报错。但是！浏览器只会承认相同id的最后一个元素。
